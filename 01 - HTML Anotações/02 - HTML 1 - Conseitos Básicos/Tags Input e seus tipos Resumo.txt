------------------------------------->> ESTRUTURA BÁSICA DE UM FORMULÁRIO <<------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Aprendendo sobre INPUT e seus TYPEs em HTML.

A TAG <label></label> colocará a mensagem no campo do form.

    FORM para o exemplo.
    <form>                                              
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------    
        É um campo texto que aceitará todo e qualquer tipo de caractere.
        -->>
        <label> Text: </label><input type="text" /><br />      
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------        
        É um campo que aceitará apenas numeros, sem formatação você poderá colocar até numeros negativos. Mas se for algo mais especifico você poderá usar os formats, como:
        Min     -->> Coloca o valor minimo de numeros
        Max     -->> Coloca o valor maximo de numeros
        Step    -->> Especifica o incremento/intervalos dos numeros (Ex: 1, 2, 3, 4, 5, 6)
        -->>
        <label> Number: </label><input type="Number" min="0" step="5" max="99" /><br /> 
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        É um campo que criará um botão.
        Value   -->> Dirá qual o valor exibido na função
        -->>
        <label> button: </label><input type="button" value="Enviar" /><br />
        -->>
        Podemos ter também a TAG button que terá seus próprios TYPEs
        -->>
        <button type="button"> Enviar </button>
        Button   -->> É um botão normal
        Submit   -->> É um botão de envio
        Reset    -->> É um botão de limpar
        Menu     -->> Abre um menu
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo scroll para valor minimo e valor maximo
        -->>
        <label> Range: </label><input type="range" min="0" max="99" value="50" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo com o tipo color
        -->>
        <label> Color: </label><input type="color" value="#ff0000" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo do tipo email
        -->>
        <label> Email: </label><input type="email" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo tipo URL
        -->>
        <label> URL: </label><input type="url" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo com o tipo date
        -->>
        <label> Date: </label><input type="date" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo tipo semana NÃO FUNCIONA NO FIREFOX
        -->>
        <label> Week: </label><input type="week" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campotipo mês
        -->>
        <label> Month: </label><input type="month" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria uma checkbox para colocar como sim ou não
        -->>
        <label> Checkbox: </label><input type="checkbox" /><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria uma radio button para selecionar uma opção
        -->>
        <label> Radio: </label><input type="radio" name="radio"/></label><input type="radio" name="radio"/><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo para enviar um arquivo
        Multiple    -->> Deixa escolher vários arquivos
        -->>
        <label> File: </label><input type="file" multiple/><br />
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
        Cria um campo de pesquisa 
        -->>
        <label> Search: </label><input type="search" /><br />
