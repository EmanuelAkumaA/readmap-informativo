Agreupamento de Seletores

Podemos aplicar a mesma regra CSS para vários seletores diferentes Para isso, podemos separar os seletores utilizando uma virgula (.).

Assim, não precisamos ficar copiando e colando o mesmo código várias vezes Nós podemos fazer um bloco de CSS e reutilizar as mesmas regras!
